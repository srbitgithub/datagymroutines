# 🚀 Novedades de IronMetric - Actualización del Día

Sigo puliendo la aplicación basándonos en el uso real. Aquí os dejo un resumen de las novedades y retoques que he implementado hoy para que IronMetric se sienta mucho más limpia, fluida y profesional.

## 🏋️‍♂️ ¡40 Ejercicios Base Nuevos!
- **Catálogo bestialmente ampliado**: He inyectado al sistema 40 de los ejercicios más comunes del gym. Ahora las cuentas nuevas arrancan con **50 ejercicios listos para usar** el primer día (Press Francés, Zancadas, Remos, Planchas...). 

## 🌍 IronMetric Más Internacional
- Varias pantallas nuevas (como la ventana de configuración del primer día o nuestra página web principal) solo hablaban español. Las he dotado de "cerebro" para que si un usuario de Estados Unidos u otro país extranjero abre la aplicación, detecte su teléfono y muestre todo automáticamente en Inglés.

## He hecho un onboarding para el usuario nuevo
- He creado un onboarding para el usuario nuevo que le guiará paso a paso por la aplicación.
- Esto aparece en cada pantalla una vez, así que si os sale, con aceptar es suficiente.
